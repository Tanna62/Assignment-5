Question-1: What are the different ways to select an element in the DOM?
There are different ways. If I need one element, I use getElementById(). If I need multiple, I use getElementsByClassName() or getElementsByTagName(). But querySelector() is easier because I can use CSS selectors, and querySelectorAll() gives all matching elements.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML gets/sets HTML + text, innerText gets only visible text, and textContent gets all text even if hidden. Example: innerHTML = "Hi" adds bold text, but innerText = "Hi" shows Hi as normal text.
Question-3: What is event delegation in the DOM?
Instead of adding events to every small element, I put one event on their parent and check event.target to know which child was clicked. This helps when new elements are added later because I don’t need to add events again. It makes code better and easier.
Question-4: What is event bubbling in the DOM?
When I click an element inside another, the event first happens on that element, then moves up to its parent, then further up. Like, if I click a button inside a div, first the button’s event runs, then the div’s. If I don’t want this, I use event.stopPropagation().
Question-5: How do you create, add, and remove elements using JavaScript?
I can create an element using document.createElement(), then add it using appendChild(). If I don’t need it anymore, I can remove it with .remove(). Example: let div = document.createElement("div"); div.innerText = "Hello"; document.body.appendChild(div); adds a div with text.
